# 015 &mdash; Hello, classes (snippet)
> illustrates basic concepts of Scala classes

## Description
Illustrates how a simple class featuring a private num and a private string you just have to type:
```scala
class MyClass(num: Int, name: String)
```

## Running the Snippet
The snippet is intended to be run from `sbt consoleQuick`. Once inside the console type:
```
scala> :load hello-classes.scala
```
